# Dataset Familiarization

## Overview

This notebook is created to explore and understand the structure and quality of the dataset before performing any advanced analysis. The objective is to document the schema of each file, inspect data types, identify missing values, and gain familiarity with the relationships between the datasets.

## Objectives

- Explore each dataset included in the project.
- Document column names and data types.
- Identify value ranges and sample records.
- Detect missing or historical data.
- Understand the purpose of each file.
- Prepare the dataset for future data analysis and machine learning tasks.

## Files Explored

The notebook examines the following files:

- **Listings**
- **Listing Summary**
- **Neighbourhoods.csv**
- **Reviews**
- **Reviews Summary**
- **Calendar**

Additional sections include:

- **Scraping Artifacts**
- **Missing Historical Data**

## Notebook Structure

### 1. Dataset Schema Documentation

For each dataset, the notebook includes:

- Column names
- Data types
- Value ranges
- Sample values
- Basic dataset information

### 2. Dataset Familiarization

The notebook helps understand:

- The purpose of each dataset
- The meaning of important columns
- Data consistency
- Potential relationships between files

### 3. Data Quality Inspection

The notebook identifies:

- Missing values
- Null entries
- Historical or unavailable data
- Data completeness
